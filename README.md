# NewsPushMonitor
NewsPushMonitor是一款可以监听手机上APP推送的通知的应用，可以帮助开发者调试自己APP的通知是否正常，或者收集统计各种APP的推送通知，包含以下功能：

- 监听APP的推送通知，提取出标题和内容
- 定期自动唤起被监听APP
- 自定义需要监听的APP列表  
- 支持提取标准通知和自定义通知内容
- 使用辅助模式，无需root和修改系统代码
